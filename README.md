小程序 wxml 转换成 h5 react 写法的 render , less rpx 转换成  rem


两个html 加一些正则 匹配



## 运行方式：
启动两个 （wxml2html  和 wxss2css）
```
npm start
```
启动 wxml2html
[转换规则](http://note.youdao.com/noteshare?id=9333e45c78abbe257c31f322b50991d7)

```
npm run start-wxml2html
```
启动 wxss2css
[转换规则](http://note.youdao.com/noteshare?id=a0a230e900a8cdff267c8a0a95237b2c)
```
npm run start-wxss2css
```
